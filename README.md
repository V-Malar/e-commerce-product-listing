# E-Commerce Product Listing — React Assignment

A React frontend demo built for the practical assignment: product listing with
category filtering, a reusable `Button` component with variants, and a custom
`useFetch` hook for API data.

## Get a live public link (no coding needed)

**Option A — CodeSandbox (fastest, ~1 minute)**
1. Go to https://codesandbox.io/
2. Click "Create" → "Import Project" (or just drag this whole folder onto the page)
3. It auto-detects Vite + React and gives you a live `https://xxxx.csb.app` URL instantly — that's your link to share.

**Option B — StackBlitz**
1. Go to https://stackblitz.com/
2. Click "New Project" → upload/drag this folder
3. It boots automatically and gives you a live `https://stackblitz.com/edit/...` preview URL.

**Option C — Vercel (best if you also want it on your GitHub profile)**
1. Push this folder to a new GitHub repo.
2. Go to https://vercel.com/ → "Add New Project" → import that repo.
3. Vercel detects Vite automatically, click Deploy, and you get a permanent `https://your-project.vercel.app` link. This also satisfies the "share your GitHub profile or live project link" part of the assignment.

## Run locally
```bash
npm install
npm run dev
```
